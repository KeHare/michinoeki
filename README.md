# 廻（めぐり）— 道の駅めぐり 北海道

北海道の道の駅128駅を効率的にめぐるためのアプリ（PWA）。「廻」は、めぐる・まわるの一字から。

## できること
- **近くの駅** — 現在地（GPS or 主要8都市）から近い順に未訪問の駅を表示。温泉・EV充電・レストラン・産直・ドッグランで絞り込み。
- **巡回プラン** — 現在地→最寄り→そこからさらに最寄り…と連鎖でつなぐ一筆書きの巡回ルートを自動生成。
- **ドライブ** — 「今から◯時間で回って戻る」の時間予算に収まる周回プランを自動生成。
- **地図** — 全128駅とプランを地図表示。マーカーから詳細・ナビへ。
- **記録** — エリア別に訪問チェック。書き出し／読み込みで機種変更にも安全。
- ナビは Googleマップに連携。各駅の営業時間・休館日・設備・マップコードを公式情報から収録。

## データ出典
- 駅一覧・営業情報・設備：[北の道の駅（公式）](https://hokkaido-michinoeki.jp/)
- 座標：公式ページ＋Wikipedia／OpenStreetMap
- 駅間の実道路距離・所要時間：[OSRM](https://project-osrm.org/)
- 地図タイル：© OpenStreetMap contributors

※本アプリは道の駅めぐりの補助を目的とした非公式ツールです。営業時間等は変更される場合があるため、公式情報もご確認ください。

## 技術メモ
- 単一HTML＋データJSのPWA（オフライン起動対応）。
- ストア公開はCapacitorでのラップを想定。
